# Web App to Mint and display NFT metadata

## Installation dependencies

```bash
yarn install
```

## Create a .env file

```bash
cp .env.example .env
```

## Run the app

```bash
yarn start
```

## Access the app

[http://localhost:3000](http://localhost:3000)
